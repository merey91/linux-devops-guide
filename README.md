# Linux DevOps Guide

This repository contains practical Linux administration knowledge and commands commonly used in DevOps, cloud infrastructure, and system operations.

The goal of this guide is to document essential Linux concepts and commands used in real-world environments such as cloud servers, CI/CD pipelines, and containerized applications.

## Topics Covered

- File System Management
- User and Group Administration
- Networking Commands
- Process Management
- Package Management
- File Permissions
- SSH Configuration
- Bash Scripting
- System Monitoring and Troubleshooting

## Example Areas

### File System
Working with directories, disk usage, and file operations.

### User Management
Creating users, managing groups, and privilege escalation.

### Networking
Diagnosing connectivity issues using commands such as:

- `ping`
- `netstat`
- `ss`
- `curl`
- `wget`

### Process Management
Monitoring and managing running processes using tools such as:

- `ps`
- `top`
- `htop`
- `kill`

### System Monitoring
Understanding system resource usage such as CPU, memory, and disk.

## Purpose

These notes are based on hands-on practice while learning DevOps and cloud engineering, and are intended as a quick reference for commonly used Linux commands in production environments.
